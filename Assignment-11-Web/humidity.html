Humidity

There did not seem to be any meaningful trends to glean from plotting humidity of various cities vs their latitudes.

One possible trend is that closer to the equator, there are fewer cities that have low humidity, as there seems there might be an empty space around 0degrees latitdue, between 0 and 60% humidity.

Note: The latitude is plotted on the Y-axis, with the humidity on the X-axis, so as to be more readable and easily interpreted for a plot about Earth's latitudes.
